# Investigation 1 -Basic Web Log Review
## Objective
To review web acess logs and understand normal traffic behaviour.
## Tool Used
Splunk
## Data Source
Web access logs (access_combined_wcookie)
## Steps Taken
- Verified web availability using wildcat search
- adjusted time range to include all available data
- filtered logs by web access sourcetype
- Observed HTTP status codes
## Findings 
- High volume of HTTP 200 responses
- Traffic appeared consistent with normal web activity
  ## Conclusion
  No suspicious activity identified during his review. This investigation helped build familiarity with Splunk and web logs.
